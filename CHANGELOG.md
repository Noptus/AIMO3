# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-02-11

### Added

- Composable AIMO3 solver with adaptive prompting, sandboxed code execution, and candidate aggregation.
- Groq/OpenAI-compatible CLI pipeline for solve, benchmark, and Kaggle submission automation.
- Geometry-specific recheck stage and selector/verifier arbitration passes.
- CI workflow, development tooling, and contributor/security documentation.

### Changed

- Improved packaging metadata and development dependencies in `pyproject.toml`.
- Expanded Makefile commands for linting, formatting, build, and repository hygiene.
